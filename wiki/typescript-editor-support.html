# Vim

# 语法高亮

  • leafgarland/typescript-vimopen in new window提供了语法文件用来高亮显示.ts和.d.ts。
  • HerringtonDarkholme/yats.vimopen in new window提供了更多语法高亮和DOM关键字。

# 语言服务工具

有两个主要的TypeScript插件：

  • Quramy/tsuquyomiopen in new window
  • clausreinke/typescript-tools.vimopen in new window

如果你想要输出时自动补全功能，你可以安装YouCompleteMeopen in new window并添加以下代码到.vimrc里，以指定哪些符号能用来触发补全功能。YouCompleteMe会调用它们各自TypeScript插件来进行语义查询。

if !exists("g:ycm_semantic_triggers")
  let g:ycm_semantic_triggers = {}
endif
let g:ycm_semantic_triggers['typescript'] = ['.']
